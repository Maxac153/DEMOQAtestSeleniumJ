#  UsersTestSelemium

### Описание

Цель проекта: получение навыков в написание автоматизированных авто тестов на Python. 
Сайт для тестирования (<a href="https://demoqa.com/">DEMOQA</a>).

## Технологии

- Java
- Selenium
- JUnit

## Состав проекта

- Файлы (src/main/java/tests) с Python тестами
- Отчет (test-output)
