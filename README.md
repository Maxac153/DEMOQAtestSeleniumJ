#  UsersTestSelemium

### Описание

Цель проекта: получение навыков в написание автоматизированных авто тестов на Java. 
Сайт для тестирования (<a href="https://demoqa.com/">DEMOQA</a>).

## Технологии

- Java
- Selenium
- TestNG
- Allure

## Состав проекта

- Файлы (src/main/java/tests) с Java тестами
- Отчет (test-output)
